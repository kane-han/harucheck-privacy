개인정보처리방침

하루체크 (HaruCheck) | 시행일: 2026년 2월 17일

1. 수집하는 정보

하루체크는 서비스 제공을 위해 아래 정보를 수집합니다.

  • 습관 데이터: 사용자가 등록한 습관 이름, 반복 요일, 완료 기록
  • 앱 사용 데이터: 화면 조회, 기능 사용 빈도, 앱 실행 횟수 (Firebase Analytics 통해 자동 수집)
  • 기기 정보: 기기 모델, OS 버전, 앱 버전 (서비스 개선 목적)
  • 광고 식별자: Google 광고 ID (광고 노출 목적, AdMob 사용)

하루체크는 이름, 이메일, 전화번호 등 개인 식별 정보를 수집하지 않습니다. 향후 클라우드 동기화 기능 추가 시 인증 정보(익명 로그인 또는 소셜 로그인)를 수집할 수 있으며, 이 경우 본 방침을 사전에 업데이트합니다.

2. 정보 사용 목적

  • 습관 기록 저장 및 통계 제공
  • 앱 성능 개선 및 오류 수정
  • 사용 패턴 분석을 통한 서비스 개선
  • 맞춤형 광고 제공 (무료 사용자)

3. 제3자 서비스

하루체크는 아래 제3자 서비스를 사용하며, 각 서비스의 개인정보처리방침을 따릅니다.

  • Google AdMob: 광고 노출 — Google 개인정보처리방침
  • Firebase Analytics: 사용 통계 — Firebase 개인정보처리방침
  • RevenueCat: 구독 관리 — RevenueCat 개인정보처리방침

4. 데이터 저장 및 보관

  • 습관 데이터는 기본적으로 사용자의 기기 내부(로컬)에만 저장됩니다.
  • 클라우드 동기화 사용 시 Firebase Firestore(서울 리전)에 암호화되어 저장됩니다.
  • 분석 데이터는 Firebase Analytics에 최대 14개월간 보관 후 자동 삭제됩니다.

5. 데이터 삭제

  • 앱 내 설정에서 모든 데이터를 삭제할 수 있습니다.
  • 앱을 삭제하면 기기에 저장된 모든 데이터가 함께 삭제됩니다.
  • 클라우드 데이터 삭제를 원할 경우 아래 연락처로 요청해 주세요.

6. 아동 보호

하루체크는 만 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 만 13세 미만 아동의 정보가 수집된 것을 인지한 경우 즉시 삭제합니다.

7. 방침 변경

본 개인정보처리방침이 변경될 경우, 앱 내 공지 또는 이 페이지를 통해 사전에 안내합니다.

Privacy Policy

HaruCheck | Effective: February 17, 2026

1. Information We Collect

HaruCheck collects the following information to provide our service:

  • Habit Data: Habit names, repeat days, and completion records you create
  • Usage Data: Screen views, feature usage frequency, app launch count (collected automatically via Firebase Analytics)
  • Device Information: Device model, OS version, app version (for service improvement)
  • Advertising Identifier: Google Advertising ID (for ad serving via AdMob)

HaruCheck does not collect personally identifiable information such as name, email, or phone number. If cloud sync features are added in the future, authentication data (anonymous or social login) may be collected, and this policy will be updated accordingly.

2. How We Use Information

  • Store habit records and provide statistics
  • Improve app performance and fix bugs
  • Analyze usage patterns to improve the service
  • Serve personalized ads (free users)

3. Third-Party Services

HaruCheck uses the following third-party services, each governed by their own privacy policies:

  • Google AdMob: Ad serving — Google Privacy Policy
  • Firebase Analytics: Usage analytics — Firebase Privacy
  • RevenueCat: Subscription management — RevenueCat Privacy Policy

4. Data Storage

  • Habit data is stored locally on your device by default.
  • When cloud sync is enabled, data is stored encrypted in Firebase Firestore (Seoul region).
  • Analytics data is retained in Firebase Analytics for up to 14 months and then automatically deleted.

5. Data Deletion

  • You can delete all data from within the app settings.
  • Uninstalling the app removes all locally stored data.
  • To request deletion of cloud data, please contact us at the email below.

6. Children's Privacy

HaruCheck does not knowingly collect personal information from children under 13. If we become aware that such information has been collected, we will delete it immediately.

7. Changes to This Policy

If this privacy policy is updated, we will notify users through in-app notices or this page in advance.
